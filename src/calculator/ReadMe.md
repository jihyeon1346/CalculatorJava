계산기 과제입니다.

파일은  필수기능 Step1,2와 도전기능을 추가한 Step3가있습니다

주요 기능
사칙연산 (+, -, *, /)
정수 및 실수 연산 지원
계산 이력 저장 및 조회
이력 삭제 기능
  
Step 1,2  에서는 int 정수로 입력값을 받는 
계산기를 만들었습니다.
while 안에 try catch를 사용해 정수를 입력받을때까지 입력구간이 반복,
연산자를 받는 문자값은 if문을 이용해 원하는 값이 되면
탈출하게 만들었습니다.
계산은 Step2파일에 분리해 캡슐화를해주고 switch문을 사용해 입력받은 연산자 문자열을 비교해
계산 후 컬렉션에 넣어 보관했습니다. 그리고 result변수안에 입력받은값들을 종합해 출력 후
계산을 종료하질 계산 내역을 볼지 선택하는 메뉴를 만들었고 내역확인시 컬렉션을 불러와 저장된
값들을 출력하고 선택지에서 이력삭제시 컬렉션의 0번째(처음 들어간 값)만 삭제되게 만들었습니다.

Step3부터는 도전기능을 추가하였습니다.
연산기능을 enum으로 각 연산자를 Enum 상수로 정의하고 추상 메서드로 연산 로직을 구현했습니다.
그리고 제너릭을 이용해 타입의 확장<T extends Number>를 사용하여
 Integer, Double 등 다양한 숫자 타입을 지원합니다.
